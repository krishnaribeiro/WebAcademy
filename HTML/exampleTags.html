História e propriedades do HTML


O HTML é uma linguagem de marcação para descrever documentos web. Nessa linguagem, pode conter imagens, textos, áudios e até vídeos. Existem várias versões do HTML, a mais atual é a HTML5, sendo utilizada para ler documentos em HTML no navegador, nele, com as formatações, é exibido apenas o conteúdo e não as TAGS. A estrutura do HTML é composta por head (cabeça) e body (corpo), onde somente o body é exibido no navegador. A grande diferença para o HTML5 é que o interpretador só entende que é ele quando é usado o DOCTYPE. Contém 2 elementos HTML: Título e paragráfo e tem sempre que fechar as tags utilizando o / . É utilizado também por convenção, as etiquetas de forma minúscula, apesar de não ter problema utilizá-las de forma maiúscula. Meta elementos tipicamente são usados para especificar descrição da página, palavras-chave, autor do documento, modificado pela última vez, e outros metadados. HTML também define elementos especiais, para definir o texto com um significado especial. HTML usa elementos para formatação de saída, como negrito ou itálico texto.
